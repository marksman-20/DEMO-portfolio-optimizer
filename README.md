# Kalpi Capital Portfolio Optimizer

## Overview
End-to-end portfolio optimization system with 9 quantitative methods implemented in Python with FastAPI backend and Streamlit UI.

## Features
- 9 portfolio optimization methods:
  1. Mean-Variance Optimization (Efficient Frontier, Max Sharpe, Min Variance)
  2. CVaR Minimization
  3. Risk Parity (Equal Risk Contribution)
  4. Tracking Error Minimization
  5. Information Ratio Maximization
  6. Kelly Criterion Optimization
  7. Sortino Ratio Maximization
  8. Omega Ratio Maximization
  9. Minimum Maximum Drawdown
- Indian stock data from Yahoo Finance (.NS suffix)
- Comprehensive performance metrics
- Interactive visualizations
